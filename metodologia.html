METODOLOGÍA


Empezamos con detalles menores para conocer a fondo la forma en que se organizó el código que se implemento en la Fase 1. Esto se hizo durante el primer mes (1° Sprint) de desarrollo de la Fase 2. Para el segundo mes, empezamos con el test MINI PLUS; con su diseño así como la inclusión del mismo a la base de datos. Tambien, la forma en organización las preguntas y como se iban a mostrar en cada modulo. En el tercer mes, se implemento el código necesario, modificaciones a la base de datos, entre otros; para la funcionalidad de los módulos que, hasta ahora, se tiene dentro del sistema. Durante la Fase 2, se hicieron arreglos menores que fueron descubiertos durante el procesos de las diversas tareas planeadas para el proyecto.
