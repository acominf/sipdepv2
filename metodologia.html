METODOLOGÍA


PRIMER MES DE TRABAJO

Durante el primer mes, empezamos con detalles menores para conocer a fondo la forma en que se organizó el código que se implemento en la Fase 1.
Durante la Fase 2, se hicieron arreglos menores que fueron descubiertos durante el procesos de las diversas tareas planeadas para el proyecto.


SEGUNDO MES DE TRABAJO

Para el segundo mes, empezamos con el test MINI PLUS; con su diseño así como la inclusión del mismo a la base de datos.
Tambien, la forma en organización las preguntas y como se iban a mostrar en cada modulo.


TERCER MES DE TRABAJO

En el tercer mes, se implemento el código necesario, modificaciones a la base de datos, entre otras diferentes modificaciones al sistema actual; para la funcionalidad de los módulos que, hasta ahora, se tiene dentro del sistema.
También, se invirtió tiempo para hacer las pruebas necesarias para ver la funcionalidad de cada módulo terminado.
